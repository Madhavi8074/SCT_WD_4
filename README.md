# Task 04 – To-Do Web Application ✅

This project is developed as part of the SkillCraft Technology Web Development Internship.

## Description
An advanced to-do web application built using MVC architecture. Supports task creation, completion tracking, deadlines, and time calculations with data persistence using LocalStorage.

## Features
- Add, edit, delete tasks
- Task deadlines and time tracking
- MVC architecture
- LocalStorage support
- Responsive UI

## Technologies Used
- HTML
- CSS
- JavaScript
